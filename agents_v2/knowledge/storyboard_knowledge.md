# 短剧分镜知识库 - 导演Agent专用

## 一、基础通用规则（强制标准）

### 1）镜头时长规范
- 单镜头上限：15秒
- 最优区间：6-12秒
- 对话镜头：长台词必须切镜拆分，一句长台词拆2个镜头
- 情绪留白镜头：2-5秒（沉默、眼神特写、停顿反应）
- 钩子定格镜头：2-3秒，片尾固定配置

### 2）镜头类型极简定义
| 镜头类型 | 定义 | 使用场景 |
|---------|------|---------|
| 全景 | 交代空间环境、人物位置关系 | 开篇、多人同场 |
| 中景 | 膝盖以上，人物肢体动作、双人对话主力镜头 | 日常对话 |
| 中近景 | 腰以上，对话主流镜头 | 对线争吵 |
| 近景 | 胸口以上，面部情绪 | 情绪爆发 |
| 特写 | 五官、手部、物品（门把手、手机、酒杯、眼神） | 关键道具、细节 |
| 侧脸特写 | 氛围感、人物内心戏 | 清醒男主、隐忍女主高频 |
| 慢推 | 情绪升级、高光台词、结尾悬念 | 觉醒段落 |

### 3）运镜通用守则
- 普通对话：固定机位，轻微缓推，禁止大幅度晃动
- 冲突升级：小幅缓慢前推
- 高光觉醒镜头：静态机位 + 轮廓光，少运镜，靠表演氛围
- 结尾钩子：慢推至人物面部特写，最后定格

### 4）光影 & 美颜标准
| 场景类型 | 光影方案 | 美颜参数 |
|---------|---------|---------|
| 颓废/压抑室内 | 低亮度、柔光、侧光 | 磨皮30-40，保留皮肤肌理 |
| 觉醒高光段落 | 增加轮廓逆光，区分前后人设反差 | 磨皮20-30，保留质感 |
| 争吵戏份 | 硬一点的侧光，强化情绪张力 | 磨皮30-40 |
| 室外白天 | 自然光均衡，避免大面积过曝 | 磨皮20-30 |

## 三、镜头切换逻辑知识库

### 说话人切换规则
```
A说完 → 切A收尾特写 → 立刻切B镜头开始说话
```

### 长文本切割规则
```
台词超过22个字 → 强制拆分两个镜头，中间插入表情/手部过渡特写
```

### 情绪拐点切镜
```
人物心态发生转变（散漫→严肃）→ 必须插入动作过渡镜头（起身、放下物品）
```

### 节奏控制
```
连续3个近景对话后 → 插入1个全景缓解视觉疲劳
```

## 五、行业通用避坑库

| 问题 | 检测规则 | 建议 |
|-----|---------|------|
| 镜头过长 | 单个镜头预估时长>15s | 预警，自动建议拆分 |
| 节奏过碎 | 连续4个特写镜头无全景/中景 | 节奏预警 |
| 缺少光影区分 | 高光段落没有光影区分前后人设 | 标注优化建议 |
| 缺少钩子 | 结尾无定格特写+留白 | 缺少钩子提醒 |
